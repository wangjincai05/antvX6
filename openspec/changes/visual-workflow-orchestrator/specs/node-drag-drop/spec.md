## ADDED Requirements

### Requirement: 节点拖拽功能
系统应支持从节点面板拖拽节点到画布，并在鼠标释放位置创建节点实例。

#### Scenario: 从面板拖拽节点到画布
- **WHEN** 用户从节点面板拖拽一个节点类型到画布区域
- **THEN** 画布应在释放位置创建一个该类型的节点实例

#### Scenario: 拖拽过程中显示预览
- **WHEN** 用户拖拽节点时
- **THEN** 系统应显示节点预览，并跟随鼠标移动

#### Scenario: 拖拽取消
- **WHEN** 用户在拖拽过程中按下 ESC 键或释放鼠标在画布外
- **THEN** 系统应取消拖拽操作，不创建节点

### Requirement: 节点类型支持
系统应支持多种节点类型的拖拽，包括开始节点、结束节点、任务节点、条件节点等。

#### Scenario: 支持多种节点类型
- **WHEN** 用户从节点面板选择不同类型的节点
- **THEN** 拖拽到画布后应创建对应类型的节点
