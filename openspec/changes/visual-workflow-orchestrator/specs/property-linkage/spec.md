## ADDED Requirements

### Requirement: 属性联动
系统应支持节点属性的联动更新，当一个节点的属性变化时，相关联的节点属性应自动更新。

#### Scenario: 属性变化联动
- **WHEN** 用户修改节点 A 的某个属性值
- **THEN** 与节点 A 关联的节点 B 的对应属性应自动更新

#### Scenario: 联动规则配置
- **WHEN** 用户在属性面板配置联动规则
- **THEN** 系统应保存规则并在属性变化时触发联动

### Requirement: 属性面板
系统应提供属性面板，用于查看和编辑选中节点的属性。

#### Scenario: 选中节点显示属性
- **WHEN** 用户选中一个节点
- **THEN** 属性面板应显示该节点的所有可配置属性

#### Scenario: 编辑属性
- **WHEN** 用户在属性面板修改属性值
- **THEN** 节点应实时更新显示

### Requirement: 数据同步
系统应支持工作流数据的导入和导出。

#### Scenario: 导出工作流
- **WHEN** 用户点击导出按钮
- **THEN** 系统应将当前工作流数据序列化为 JSON 格式并下载

#### Scenario: 导入工作流
- **WHEN** 用户选择一个 JSON 文件并点击导入
- **THEN** 系统应解析文件并在画布上还原工作流
