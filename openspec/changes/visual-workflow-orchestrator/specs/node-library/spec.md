## ADDED Requirements

### Requirement: 节点库管理
系统应提供可扩展的节点库，支持多种类型的工作流节点。

#### Scenario: 查看节点库
- **WHEN** 用户打开节点面板
- **THEN** 应显示所有可用的节点类型

#### Scenario: 搜索节点
- **WHEN** 用户在节点面板输入搜索关键词
- **THEN** 应过滤显示匹配的节点类型

### Requirement: 节点类型定义
系统应支持自定义节点类型的定义和注册。

#### Scenario: 注册新节点类型
- **WHEN** 开发人员通过 API 注册新节点类型
- **THEN** 该节点类型应显示在节点面板中

### Requirement: 节点模板
系统应支持节点模板的保存和复用。

#### Scenario: 保存节点模板
- **WHEN** 用户配置好节点属性后点击保存模板
- **THEN** 节点配置应被保存为模板

#### Scenario: 使用模板创建节点
- **WHEN** 用户从模板列表选择一个模板
- **THEN** 应使用模板配置创建节点
